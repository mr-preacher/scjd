Server

To start application in server mode user must gain access to application directory and type

java -jar runme.jar server

then press Enter.

Server window will be displayed with popuped server options dialog.

Option Description
Database path Local path to database file
Server port Port number server listens
Lock timeout Amount of time in milliseconds record can be locked

User must enter appropriate option values and press OK.

If options is not valid text fields of options have pink color. Text fields tooltips show tips which text values are required.

As soon as user press OK server starts.

User can restart server with new server options at any time. For this user have to click option menu.

Status bar of server window shows current server status.

If some error occurs error dialog displayed on top of server window.

Server has text area to show log of server.
