Client

To start application in client mode user must gain access to application directory and type

java -jar runme.jar

then press Enter.

Client window will be displayed with popuped client options dialog.

Option Description
Server host Host name or IP address of server
Server port Port number server listens

User must enter appropriate option values and press OK.

If options is not valid text fields of options have pink color. Text fields tooltips show tips which text values are required.

As soon as user press OK client connects to the server.

User can reconnect client with new client options at any time. For this user have to click option menu.

Status bar of client window shows current client status.

If some error occurs error dialog displayed on top of client window.

There are many operations available in client mode.

Operation Description
Book Books or unbooks record selected in table
Create Creates new record in database
Edit Edits record record selected in table
Delete Deletes record selected in table
Find Finds records in database

User can use context or main menu to operate database.

If user chooses find menu dialog appears.

Find dialog popups if user press Yes.

Result shown in table.

To book record choose Book menu.

And then record in table updated.
