Standalone

To start application in standalone mode user must gain access to application directory and type

java -jar runme.jar alone

then press Enter.

Client window will be displayed with popuped standalone options dialog.

Option Description
Database path Local path to database file
Lock timeout Amount of time in milliseconds record can be locked

User must enter appropriate option values and press OK.

If options is not valid text fields of options have pink color. Text fields tooltips show tips which text values are required.

As soon as user press OK standalone read file from local hard drive.

User can define another database file at any time. For this user have to click option menu.

Status bar of client window shows current standalone status.

If some error occurs error dialog displayed on top of client window.

There are many operations available in client mode.

Operation Description
Book Books or unbooks record selected in table
Create Creates new record in database
Edit Edits record record selected in table
Delete Deletes record selected in table
Find Finds records in database

User can use context or main menu to operate database.

Standalone client supports all commands that network client does. For details see network client documentation.
