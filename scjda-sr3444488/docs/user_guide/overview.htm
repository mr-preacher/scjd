Overview

File scjda-sr3444488.jar contains performed task for SCJD certification.

To extract files from jar file into current directory user can use command line:

jar -xf scjda-sr3444488.jar

SCJD application has three modes (also knows as types). Each of them uses specific command of console to start.

Mode Description Command
Server Network server mode java -jar runme.jar server
Client Network client mode java -jar runme.jar
Standalone Local standalone mode java -jar runme.jar alone

These commands must be typed when user is in application directory.

Application directory is the directory that contains runme.jar file and suncertify.properties file.

To gain access to application directory user can use shell command cd.
